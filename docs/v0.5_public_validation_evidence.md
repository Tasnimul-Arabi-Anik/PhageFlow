## Commands

Single-genome validation:

```bash
nextflow run main.nf \
  --input "$EX/manifests/single_samplesheet.tsv" \
  --pangenome_method none \
  --outdir "$EX/results/public_single"

bash bin/phageflow validate \
  --outdir "$EX/results/public_single"
```

Small-cohort validation:

```bash
nextflow run main.nf \
  --input "$EX/manifests/cohort_samplesheet.tsv" \
  --pangenome_method mmseqs \
  --outdir "$EX/results/public_cohort"

bash bin/phageflow validate \
  --outdir "$EX/results/public_cohort" \
  --require-pangenome-rows
```

Completed-run utilities:

```bash
bash bin/phageflow summarize --outdir "$EX/results/public_single" --output "$EX/manifests/public_single_summary.json"
bash bin/phageflow package --outdir "$EX/results/public_single" --output "$EX/packages/public_single_phageflow_package.tar.gz" --force
bash bin/phageflow summarize --outdir "$EX/results/public_cohort" --output "$EX/manifests/public_cohort_summary.json"
bash bin/phageflow package --outdir "$EX/results/public_cohort" --output "$EX/packages/public_cohort_phageflow_package.tar.gz" --force
```
